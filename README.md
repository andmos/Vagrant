Vagrant
=======

My Vagrant template with some useful scripts for quick setup.
Forwords 8080 to 80 for easy web-development environment.
 
    sudo apt-get install -y virtualbox vagrant
    git clone 
    vagrant up

Current tested setup:
OSX:

    Vagrant 1.2.7
    Virtualbox 4.1.18  

Debian:

    Kernel 3.14
    Vagrant 1.2.7
    Virtalbox 4.2.24
