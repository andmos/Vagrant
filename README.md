Vagrant
=======

My Vagrant template with some useful scripts for quick setup.
Forwords 8080 to 80 for easy web-testing environment.
Just: 

sudo apt-get install -y virtualbox  ; sudo apt-get install -y Vagrant

git clone 

vagrant up

done.

Current tested setup:
Vagrant v. 1.2.7
Virtualbox 4.1.18
